# 🚀 Installation & Setup

### Prerequisites

* Node.js 18+
* npm or yarn

---

# 1️⃣ Clone the Repository

```bash
git clone https://github.com/kishorekrrish3/my-portfolio.git
cd my-portfolio
```

---

# 2️⃣ Install Dependencies

```bash
npm install
```

---

# 3️⃣ Run the Development Server

```bash
npm run dev
```

---

# 🌐 Local Development

The application will start at:

```
http://localhost:3000
```
